# GappyAI

GappyAI is an AI-powered job application management platform designed to simplify the internship and job hunting process. It brings application tracking and AI-assisted career tools together in one workspace, making it easier to stay organized and improve application quality.

Built as a full-stack web application using the MERN stack with AI integration.

---

## Deployed Link
https://gappy-ai-two.vercel.app/

## ✨ Features

- 📋 Track job and internship applications
- 📊 Dashboard with application statistics
- 🤖 AI-powered resume analysis
- ✍️ AI-generated resume feedback and suggestions
- 🔎 Search and manage applications
- 📱 Responsive UI for desktop and mobile

## Why GappyAI?

Job searching usually involves juggling multiple spreadsheets, documents, and websites. GappyAI aims to bring everything into a single workspace by combining application tracking with AI-powered career assistance.

Instead of switching between tools, users can manage applications, improve resumes, and prepare for opportunities from one dashboard.
